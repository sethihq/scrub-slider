# Changelog

## 0.1.0 (2026-03-30)

### Features
- Slider component with Radix UI primitives
- Scrub sounds via Web Audio API (base64-encoded, no external files)
- Haptic feedback via web-haptics library
- Animated value display with NumberFlow
- Hover value chip (top or bottom position)
- Optional sound and haptics via props
- Full dark mode support with CSS custom properties
- Accessible — built on Radix UI slider primitive

### Demo
- Interactive playground with tweakable parameters
- Package manager install tabs (npm/pnpm/yarn/bun)
- Framework usage examples (React/TypeScript/Vue/Svelte)
- Props reference table
- CSS tokens reference
